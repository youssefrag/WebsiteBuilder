# Website Builder

## Choose between different component types: Title/ picture etc

## Have the option to choose between a few different customizations for each component

## Drag and drop the components into the website

## Save website as JSON object once completed

## Save to PSQL Database

## Create function for opening newly generated website to a new tab

## Create different layout for edit and view

## Be able to edit components and save them again
